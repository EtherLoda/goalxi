2. 对抗系统（Sigmoid） 2. The Duel System (Sigmoid)

所有对抗结果都使用 Sigmoid 对抗函数。包括中场争夺、进攻推进、射门、定位球，全部基于此。

All contest outcomes use a sigmoid duel function. Used for midfield battles, attack pushes, shots, set pieces — everything.

P = 1 / (1 + exp(-(ratio - 1 - offset/100) × k))
参数 Param 含义 Meaning
ratio valA / valB 进攻方 / 防守方 attacker value / defender value
offset 偏置值 正偏置 = 进攻方更容易赢；负偏置 = 进攻方更难赢 Positive = easier for attacker; negative = harder
k 敏感度 k 越高，相同 ratio 差时结果越极端 Higher k = more decisive at same ratio difference
P 概率 进攻方 valA 获胜的概率 Probability that valA (attacker) wins

3. 核心概念 3. Core Concepts

3.1 三路体系 Three Lanes

场地分为三路：左路、中路、右路。所有进攻强度、防守强度、控球强度都按路计算。每次关键时刻选择其中一路。

The pitch is divided into three lanes: left, center, right. All attack power, defense power, and possession strength are calculated per-lane. One lane is selected per key moment.

3.2 路强度计算 Lane Strength

每队每路的强度 = 该路所有球员贡献值的加权和：

Each team's strength in each lane is a weighted sum of player contributions:

  • 进攻强度 = 该球员在左路进攻贡献之和
  • Attack strength = sum of each player's attack contribution in that lane
  • 防守强度 = 该球员在左路防守贡献之和
  • Defense strength = sum of each player's defense contribution in that lane
  • 控球强度 = 该球员在左路控球贡献之和
  • Possession strength = sum of each player's possession contribution in that lane

球员贡献由 AttributeCalculator 根据球员属性、位置契合度、体能/状态修正计算。

Player contributions are calculated by AttributeCalculator based on player attributes, position fit, stamina/fitness multiplier, and form.

3.3 控球方 vs 防守方 Possession Team vs Defending Team

中场争夺之后，胜方成为 possessionTeam（控球方），负方成为 defendingTeam（防守方）。后续所有计算（路强度、进攻类型、推进判定）都基于此身份。

After the midfield battle, the winner becomes possessionTeam, the loser becomes defendingTeam. All subsequent calculations (lane strength, attack type, push) use this designation.

拦截反击机制（Counter-attack / Intercept） Counter-attack (Intercept) Mechanism

当 defRatio = defPower / attPower > 1.3 时，有概率发生拦截并发动反击：

When defRatio = defPower / attPower > 1.3, there's a chance the defender intercepts and launches a counter:

  • 拦截概率: min(0.25, (defRatio - 1.3) × 0.1)
  • Intercept chance: min(0.25, (defRatio - 1.3) × 0.1)
  • 触发时：控球方 ↔ 防守方立即互换
  • If triggered: possessionTeam ↔ defendingTeam swap immediately
  • 新进攻方用自己当前路的进攻强度和防线加成重新计算 effectiveAttPower
  • New attacking team recalculates effectiveAttPower with its own lane strength + defensive line modifier
  • 反击加成: attPower × (1 + min(0.30, (defRatio - 1.3) × 0.15 × (1/counterVuln)))
  • Counter bonus: attPower × (1 + min(0.30, (defRatio - 1.3) × 0.15 × (1/counterVuln)))
  • SLOW 队: counterVuln=1.3 → 加成 = 1/1.3 = 0.77×（被反击时更安全）
  • SLOW team: counterVuln=1.3 → bonus = 1/1.3 = 0.77× (less exposed when countered)
  • FAST 队: counterVuln=0.7 → 加成 = 1/0.7 = 1.43×（被反击时更危险）
  • FAST team: counterVuln=0.7 → bonus = 1/0.7 = 1.43× (more exposed when countered)
  • pushSuccess = true → 直接进入射门（不再重复判定）
  • pushSuccess = true → goes directly to shot (no second resolveDuel)
  • interceptTriggered 时，重新从新进攻方选射手
  • When interceptTriggered, shooter is re-selected from the new attacking team

10. 体能系统 10. Condition System (Stamina & Fitness)

每分钟 Team.updateCondition() 衰减球员体能并重新计算 condition multiplier。该修正值应用于球员所有贡献值。

Every minute, Team.updateCondition() decays player fitness and recalculates a condition multiplier. This multiplier scales all of a player's contribution values.

multiplier = f(fitness, stamina, form, experience)
  • 体能衰减: 比赛期间每分钟约 0.5%。中场休息（46分钟）时部分恢复。
  • Stamina decay: ~0.5% per minute during play. Partial recovery at halftime (46).
  • Fitness: 从 1.0 开始，随体能衰减。代表身体准备度。
  • Fitness: Starts at 1.0, decays with stamina. Represents physical readiness.
  • Form: 静态的比赛间修正值（1–10范围），代表近期表现。
  • Form: Static match-to-match modifier (1–10 scale). Represents recent performance.
  • Experience: 降低低体能/低 fitness 的负面影响。
  • Experience: Reduces the negative impact of low stamina/fitness.
  • Condition multiplier: 应用于所有贡献计算。最大疲劳时约 0.7。
  • Condition multiplier: Applied to all contribution calculations. Min ~0.7 at full fatigue.

被以下模块使用：AttributeCalculator 在计算路强度、射门评分、定位球得分时都会应用此修正。

Used by: AttributeCalculator when calculating lane strengths, shot ratings, set piece scores.
